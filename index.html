Exercice 1 : Min/Max sur Ellipso√Øde

Minimiser \( f(x) = (x_1-2)^2 + x_2^2 + x_3^2 \) S.T. \( h(x) = x_1^2 + 2x_2^2 + 3x_3^2 - 2 = 0 \).

1. Formulation du Lagrangien \( L \)

$$ L(x, \lambda) = (x_1-2)^2 + x_2^2 + x_3^2 + \lambda(x_1^2 + 2x_2^2 + 3x_3^2 - 2) $$

2. Conditions du Premier Ordre (FONC) - R√©solution Alg√©brique Compl√®te

R√©solution de \( \nabla L = 0 \):

**\( \frac{\partial L}{\partial x_1} = 0 \)** : \( 2(x_1 - 2) + 2\lambda x_1 = 0 \implies 2x_1(1 + \lambda) = 4 \implies x_1 = \frac{2}{1+\lambda} \quad (\mathbf{E1}) \)
**\( \frac{\partial L}{\partial x_2} = 0 \)** : \( 2x_2 + 4\lambda x_2 = 0 \implies 2x_2(1 + 2\lambda) = 0 \quad (\mathbf{E2}) \)
**\( \frac{\partial L}{\partial x_3} = 0 \)** : \( 2x_3 + 6\lambda x_3 = 0 \implies 2x_3(1 + 3\lambda) = 0 \quad (\mathbf{E3}) \)
**\( \frac{\partial L}{\partial \lambda} = 0 \)** : \( x_1^2 + 2x_2^2 + 3x_3^2 - 2 = 0 \quad (\mathbf{E4}) \)

L'analyse des cas possibles (E2 et E3) montre que seule la solution **\( x_2 = 0 \) et \( x_3 = 0 \)** est compatible avec (E1) et (E4). Substitution dans (E4) donne \( x_1 = \pm\sqrt{2} \).

Les deux points r√©guliers (stationnaires) sont :

**P1 :** \( (\sqrt{2}, 0, 0) \) avec \( \lambda_1 = \frac{2}{\sqrt{2}} - 1 = \sqrt{2} - 1 \approx 0.414 \)
**P2 :** \( (-\sqrt{2}, 0, 0) \) avec \( \lambda_2 = \frac{2}{-\sqrt{2}} - 1 = -\sqrt{2} - 1 \approx -2.414 \)

3. Conditions du Second Ordre (SONC) et Nature des Points

La Hessienne \( H_L = \nabla_{xx}^2 L \) est diagonale :

$$ H_L = \begin{pmatrix} 2+2\lambda & 0 & 0 \\ 0 & 2+4\lambda & 0 \\ 0 & 0 & 2+6\lambda \end{pmatrix} $$

Pour **P1 (\(\lambda = 0.414\))** : Tous les termes diagonaux (valeurs propres) sont strictement positifs (\( > 0 \)). \( H_L \) est **D√©finie Positive**.

**Conclusion P1 :** Point de **Minimum Local**.

Pour **P2 (\(\lambda = -2.414\))** : Tous les termes diagonaux (valeurs propres) sont strictement n√©gatifs (\( < 0 \)). \( H_L \) est **D√©finie N√©gative**.

**Conclusion P2 :** Point de **Maximum Local**.

4. Code de V√©rification (MATLAB et Python)

Script MATLAB (R√©solution Compl√®te et Analyse de la Hessienne)


% EXERCICE 1 : R√©solution Symbolique et Analyse
syms x1 x2 x3 lambda real

f = (x1 - 2)^2 + x2^2 + x3^2;
h = x1^2 + 2*x2^2 + 3*x3^2 - 2;
L = f + lambda * h;

% 1. R√©solution des FONC
grad_L = jacobian(L, [x1, x2, x3, lambda]);
sol = solve(grad_L == 0, [x1, x2, x3, lambda], 'Real', true);

disp('--- Points Critiques (x1, x2, x3, lambda) ---');
disp(double([sol.x1, sol.x2, sol.x3, sol.lambda]));

% 2. Analyse SONC (Hessienne)
H = hessian(L, [x1, x2, x3]); 
H_P1 = double(subs(H, [x1, x2, x3, lambda], [sol.x1(1), sol.x2(1), sol.x3(1), sol.lambda(1)]));
fprintf('\nValeurs propres P1: %.4f, %.4f, %.4f --> MINIMUM\n', eig(H_P1));

Exercice 4 : Maximisation du Volume

Maximiser \( V = XYZ \). Contraintes : \( 2(XY + YZ + XZ) = 2700 \) et \( X = 2Y \).

1. R√©duction du Probl√®me (Substitution Alg√©brique)

On substitue \( X = 2Y \) dans les fonctions :

**Objectif (R√©duit) :** \( V(Y, Z) = (2Y) \cdot Y \cdot Z = 2Y^2Z \)
**Contrainte \( h(Y, Z) \):** \( 2((2Y)Y + YZ + (2Y)Z) = 2700 \) $$ 2(2Y^2 + 3YZ) = 2700 \implies 2Y^2 + 3YZ - 1350 = 0 $$

2. Lagrangien et FONC (M√©thode Min \(-V\))

Pour maximiser \( V \), on minimise \( -V \). Lagrangien : \( L(Y, Z, \lambda) = -2Y^2Z + \lambda(2Y^2 + 3YZ - 1350) \)

R√©solution de \( \nabla L = 0 \):

**\( \frac{\partial L}{\partial Y} = 0 \)** : \( -4YZ + \lambda(4Y + 3Z) = 0 \quad (\mathbf{E1}) \)
**\( \frac{\partial L}{\partial Z} = 0 \)** : \( -2Y^2 + 3\lambda Y = 0 \implies \lambda = \frac{2Y}{3} \quad (\mathbf{E2}) \)

3. R√©solution Compl√®te des Dimensions

Substitution de \(\lambda\) dans (E1) (en divisant par \( Y \neq 0 \)) :

$$ -4Z + \left(\frac{2}{3}\right)(4Y + 3Z) = 0 $$ $$ -12Z + 2(4Y + 3Z) = 0 \implies -12Z + 8Y + 6Z = 0 $$ $$ 8Y = 6Z \implies \mathbf{Z = \frac{4}{3}Y} $$

Substitution de \( Z \) dans la contrainte \( 2Y^2 + 3YZ = 1350 \):

$$ 2Y^2 + 3Y\left(\frac{4}{3}Y\right) = 1350 \implies 2Y^2 + 4Y^2 = 1350 $$ $$ 6Y^2 = 1350 \implies Y^2 = 225 \implies Y = 15 $$
**Dimensions Optimales :** $$\mathbf{Y = 15 \text{ cm}}$$ $$\mathbf{X} = 2(15) = \mathbf{30 \text{ cm}}$$ $$\mathbf{Z} = \frac{4}{3}(15) = \mathbf{20 \text{ cm}}$$ **Volume Maximal :** \( V = 30 \times 15 \times 20 = \mathbf{9000 \text{ cm}^3} \)

4. Code de V√©rification (MATLAB)

Script MATLAB (R√©solution Symbolique)


% EXERCICE 4 : R√©solution Symbolique du Volume Maximum
syms Y Z lambda real

% D√©finition des fonctions r√©duites
Volume = 2 * Y^2 * Z;
f_opt = -Volume;  % Minimisation de l'oppos√© pour la maximisation
h = 2*Y^2 + 3*Y*Z - 1350;

L = f_opt + lambda * h;

% R√©solution des FONC
grad_L = jacobian(L, [Y, Z, lambda]);
assume(Y > 0); % Les dimensions sont positives
sol = solve(grad_L == 0, [Y, Z, lambda], 'Real', true);

% Calcul et affichage des r√©sultats finaux
Y_res = double(sol.Y(1));
Z_res = double(sol.Z(1));
X_res = 2 * Y_res;
V_max = double(subs(Volume, [Y, Z], [Y_res, Z_res]));

disp('--- Dimensions Optimales (X, Y, Z) ---');
disp([X_res, Y_res, Z_res]);
fprintf('Volume Maximal : %.2f cm^3\n', V_max);
